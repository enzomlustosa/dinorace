# About
This project was made for a college assignment where visitors could come to campus and interact with works created by students.
The game was built using JavaScript, Vite, and Kaplay. It was also my first time ever using Kaplay, so a large part of the development process involved learning the library while actively working on the project itself.
I programmed the gameplay systems and mechanics and also made the sprites and visual assets used in the game.
Even with the short development time and unfamiliarity with the new library, the project became a fully playable game that was successfully presented during the event.

## What was used?
- JavaScript
- Vite
- Kaplay.js

## Features

- Character selection screen
- 2-player movement and controls
- Sprites and assets
- Scene management
- Fully playable game

## What I Learnt
- Basics of the Kaplay library
- Project structure with Vite
- Node.js project workflow
- Rapid prototyping
- Debugging while learning a new library
- Learning to work with sparse or limited documentation
- Experimenting with unfamiliar APIs and debugging through trial and error
- Managing a game project and quickly learning a new library under a short deadline


## Building The Project
If you want to run the game locally instead:

```bash
git clone https://github.com/enzomlustosa/dinorace.git
cd dinorace
npm install
npm run dev
```

Then open the localhost URL displayed in the terminal.
